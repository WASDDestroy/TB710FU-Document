TB710FU（联想小新Pad Pro GT）技术参考文档
--------------------------------------------

### 1. 文档概述

* **适用设备**：联想小新Pad Pro GT（型号 TB710FU），搭载 **六核骁龙 8 Gen 3（SM8650Q-AB）**
* **操作系统**：出厂系统为 **ZUXOS**（联想官方 Android 系统）
* **目标读者**：同前，但补充说明：
  * 若用户缺乏基础计算机操作能力（如文件管理、命令行使用等），建议先学习 [《你缺失的那门计算机课》](https://www.criwits.top/missing/?spm=a2ty_o01.29997173.0.0.737b5171neDLfF)（零门槛入门课程，涵盖文件操作、软件安装、AI 与安全等）

* * *

### 2. 基础概念与术语

| 术语                              | 说明                                                                                                           |
| ------------------------------- | ------------------------------------------------------------------------------------------------------------ |
| **9008 模式**                     | 高通 EDL 模式，可通过工程线、`adb reboot edl` 或组合键进入；需加载引导文件 `xbl_s_devprg_ns.melf`（来自官方国际版刷机包，**已验证适用于所有 TB710FU 设备**）  |
| **槽位（Slot）**                    | 采用 **VAB（Virtual A/B）** 架构，支持无缝更新但不双倍占用存储；刷机时需确认活动槽位（A/B）                                                    |
| **GKI（Generic Kernel Image）**   | 自出厂搭载 Android 13 的设备（对应Linux Kernel Version 5.10）起，内核与 vendor 解耦；KernelSU 通过**替换 `boot` 分区**实现 Root；**推荐方式** |
| **LKM（Loadable Kernel Module）** | 通过修补 `init_boot` + `vbmeta` 实现 Root，**不替换原内核**；需与 ZUXOS 版本严格匹配                                               |
| **AVB 2.0**                     | 启动时验证分区签名；未签名镜像将被拒绝启动；**所有非官方镜像必须使用已签名版本**                                                                   |
| **动态分区 / super**                | `system`、`vendor`、`product` 等逻辑分区封装于 `super`；使用 9008 工具刷写时需配合 `rawprogram.xml` + `patch.xml`                 |

> ⚠️ **Bootloader 不可官方解锁**，所有修改依赖已签名镜像或 9008 直写。

* * *

### 3. 镜像命名规范

#### 3.1 GKI 镜像命名规则

格式：  
`android<Android版本>-<Linux内核版本>_<编译日期>-boot-<Root实现>-<KernelSU版本>.img`

示例：  
`android14-6.1.112_2024-11-boot-KernelSU-2.1.2.img`

* `2024-11`：Google 内核编译时间
* `2.1.2`：KernelSU 版本号

#### 3.2 LKM 镜像命名规则

格式：  
`<Root实现>_LKM_<KernelSU版本>_<ZUXOS版本>.7z`

示例：  
`KernelSU_LKM_12123_326.7z`

* `326`：ZUXOS 系统版本号（如 1.1.04.326）
* 必须与当前系统版本一致

> 📌 **建议优先使用 GKI**：LKM 需频繁重制，维护成本高，对普通用户不友好。

### 4. 关键工具与文件说明

| 项目            | 说明                                                        |
| ------------- | --------------------------------------------------------- |
| **“匣”刷机工具**   | 推荐版本：0.4.2.913（教程基于此版）；部分用户报告刷 `super.img` 时卡 100%，重试即可解决 |

### 5. 调度与模块兼容性

* **芯片确认**：TB710FU 搭载 **六核骁龙 8 Gen 3（SM8650Q-AB）**，非完整八核版
* **调度模块**：Uperf、Scene 等主流模块**已适配且运行稳定**
* **注意**：不建议使用 A-SOUL 线程优化等实验性子模块

* * *

### 6. 安全与风险说明

* **eFUSE 风险**：目前 **无证据** 表明联想通过熔断 eFUSE 限制 9008 模式（高通平台暂无类似联发科的广泛封锁案例）
* **保修状态**：因 Bootloader 未解锁，设备仍享有官方保修（但若自行拆机或触发 FRP 锁可能影响）

* * *

### 7. 系统与 ROM 说明

* **ZUXOS**：联想官方系统名称，即出厂 Android 系统
* **第三方 ROM**：ColorOS 15/16、HyperOS 2.0/3.0 均为基于 ZUXOS 或其他厂商系统**重打包的 9008 全量镜像**
* **互刷限制**：ColorOS 15 与 HyperOS 2.0 **不能互刷**，会触发 AVB 校验失败
