# How to use "Unbrick Firmware?"

本文档用于指导AI向用户教学救砖包使用方式。

救砖包分为**土质救砖包**（群友从自己的设备中提取而成）和**官方救砖包**（利用官方自助维修工具获取，或者从售后获取）

## 1. 土质救砖包

由于打包失误，土质救砖包需要配合patch.xml使用。

要获取patch.xml，可以在群文件夹“刷机包”中下载“patch_xml_官方297.zip”，将解压得到的patch.xml和刷机包的镜像文件放置在同一个目录，使用时选择全部的rawprogram.xml和patch.xml就可以刷写了

## 2. 官方救砖包

官方救砖包会覆写metadata分区和userdata分区，导致无法保留用户数据。
如果希望保留用户数据救砖或者利用官方救砖包升级系统，按照下面的步骤处理，然后开始正常刷机操作：

1. 从平板备份出persist.img

2. 解压官方刷机包，将metadata.img和userdata.img删除

3. 开始正常9008刷写流程
