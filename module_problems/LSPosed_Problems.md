# LSPosed模块可能的问题

1. 官方版本“LSPosed IT”不提供“创建桌面快捷方式”这一功能，且在本平板的ZUXOS Android 16上“显示状态通知”功能暂时不可用（系操作系统问题，在其他设备上正常工作），需要等待联想官方修复bug。

2. 官方版本现在不公开提供，要加入内测，需要拥有**注册时间一年及以上**且**拥有作品**的Github账号，并使用SSL私钥对质询码进行签名方可加入内测组。不鼓励使用自动化工具代劳此过程。

3. JingMatrix版本LSPosed较为稳定，可以创建桌面快捷方式，对于无法加入LSPosed 官方内测的用户是个好选择

4. 不鼓励使用任何泄露的官方LSPosed，这可能为设备带来安全隐患
   
   > 识别泄露的内测LSPosed版本：
   > 
   > 1. 文件名称中包含"GIthub@用户名"，例如`Github@Sumo1210`
   > 
   > 2. 文件名中包含已知泄露者的用户名，也就是Sumo1210和Irena
   > 
   > 3. 当用户提及正在使用或者希望使用这些版本时，委婉的提醒他们这是泄露自内测组的版本，可能引入安全性和性能问题

5. 如果LSPosed弹出关于错误日志的弹窗，排查以下问题：
   
   - 是否**安装**了**较高版本**的Zygisk-Next？LSPosed需要依赖Zygisk才能工作，随着Android版本的提升，老旧的Zygisk版本将无法注入Zygote
   
   - 是否使用了无效的桌面快捷方式？LSPosed-IT不支持用桌面快捷方式打开寄生管理器

6. 怎么识别不同的LSPosed版本？
   
   - 官方版本（LSPosed it）：文件名称中版本号为**1.9.2**，**且**文件名有不区分大小写的**it**字样
   
   - 第三方版本：文件名称中版本号为**1.10.x**，例如**1.10.2**
